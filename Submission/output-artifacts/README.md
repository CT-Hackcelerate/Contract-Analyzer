# Output Artifacts (proof of a completed run)

After a clean run, place here:

- The generated dashboard HTML file(s) copied from `../../output/`.
- Screenshots of each dashboard page (see `../03_Demo_Script.md` checklist).
- A copy of `../../output/audit.log` (PII-masked: events, tokens, cost, warnings).
- Optionally `../../output/history.json` (the saved analysis record).

These artifacts demonstrate that the agent ran end-to-end and produced a clear, correct, polished deliverable.
